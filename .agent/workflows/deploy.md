---
description: How to deploy the Laszon Catalogue to Firebase Hosting
---

To deploy your catalogue to the live website, follow these steps in your terminal:

1. **Install Firebase CLI** (If you haven't already):
   ```powershell
   npm install -g firebase-tools
   ```

2. **Login to Firebase**:
   ```powershell
   firebase login
   ```

3. **Deploy the Site**:
   // turbo
   ```powershell
   firebase deploy
   ```

Your website will be live at: [https://laszon-uae-catalogue.web.app](https://laszon-uae-catalogue.web.app)
